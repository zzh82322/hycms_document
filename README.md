###   什么人会选择建站系统做网站呢？
 - 没有IT部门支撑的企业。
 - 懂一些简单的网页设计的个人。
 - 小微企业不想投入太多精力，金钱。


### 什么人在维护网站内容？
- 小企业的市场人员
- 行政前台/助理
- 个体老板
   

市场上有很多建站程序，功能强一些的操作过于复杂（如：Wordpress），有那些精力不如直接学习编程开发一套，自己也提升了技能又掌握了核心代码。
“华云CMS建站系统”从这两个问题点出发，再结合“企业实际业务”，打造出为企业长远发展，不断提供新功能的对外互联网系统。
> ps: 对外互联网系统是针对客户使用，如：企业官网，小程序，APP，公众号等等.
>     企业内部系统如erp,oa等等由于信息安全不对外使用。
